## v1.0.0 (Dragon Throne) — 2026-03-24

---

## 概述

OpenReign Pro 首个正式版本，基于 OpenClaw Gateway 构建的三省六部治理架构。

## 核心特性

### 🏛️ 三省六部架构
- **太子 (Taizi)**: 任务分发中枢，智能判断任务复杂度
- **中书省 (Zhongshu)**: 决策规划与任务拆解
- **门下省 (Menxia)**: 审核监督与风险管控
- **尚书省 (Shangshu)**: 执行调度与进度跟踪
- **六部 (Liubu)**: 吏部、兵部、户部、礼部、刑部、工部各司其职

### 🔄 智能任务分流
- 简单任务 (≤3) 直接调用本地工具
- 复杂任务 (≥4) 进入完整朝廷流程
- 自动复杂度评估

### 🛡️ 死循环防护
- 最大迭代次数: 50
- 最大调用深度: 10
- 自动检测重复调用、循环依赖

### 🧠 记忆分级
- L0 临时: 单次会话
- L1 会话: 24小时
- L2 短期: 7天
- L3 长期: 365天
- L4 永久: 无期限

### 🔐 严格权限控制
- 防止越级调用
- 部门间调用限制
- 违规自动阻断

### 📊 Dashboard 看板
- 任务实时监控
- 部门状态查看
- 记忆分级浏览
- 技能注册表管理
- 任务中断控制

## 安装

```bash
cd openreign-pro-v1.2
./scripts/install.sh
```

## 启动

```bash
# 手动启动
~/.openreign/bin/openreign

# 或后台服务
launchctl load ~/Library/LaunchAgents/ai.openreign.pro.plist
```

## 访问

- Dashboard: http://localhost:18790
- OpenClaw: http://localhost:18789

## 已知限制

1. 需要 OpenClaw Gateway 预先运行
2. Dashboard 需要 Node.js 18+
3. 工部功能需要 Docker
